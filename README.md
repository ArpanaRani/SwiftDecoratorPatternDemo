# CoffeApp (Decorator Design pattern) 
A SwiftUI-based demo project that demonstrates the Decorator Design Pattern using a coffee customization system. Users can select a base coffee and dynamically enhance it with add-ons like Milk, Sugar, and Cream, with price and ingredient updates.

## Architecture Overview
- SwiftUI for UI layer
- MVVM ViewModel for state handling
- Protocol-based abstraction for Coffee types
- Decorator Pattern for dynamic composition
